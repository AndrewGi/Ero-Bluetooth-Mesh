# Ero-Bluetooth-Mesh
Bluetooth mesh stack for use in ero system. May be useful for others for reference

python module named bt_mesh

See better_mesh_ctl.py for an example.

## Features Supported:
- Hardware Agnostic Bearer Layer
- Custom Sockets Proxy
- Network Layer
- Transport Layer
- Access Layer

## Features Planned:
- Message Cache
- Model Layer
- Friends
- Low Power?
- Relay
